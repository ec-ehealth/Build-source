ValueSet civil state

Codes supported by eHealth Platform differentiating types of civil state. This valueset supports the Belgian federal FHIR profiling effort. Whenever possible add a code from http://terminology.hl7.org/CodeSystem/v3-MaritalStatus for international interoperability but also use https://www.ehealth.fgov.be/standards/fhir/CodeSystem/CD-CIVILSTATE for the Belgian specific code.

  • Include codes from http://terminology.hl7.org/CodeSystem/v3-MaritalStatus where concept is-a A
  • Include codes from http://terminology.hl7.org/CodeSystem/v3-MaritalStatus where concept is-a D
  • Include codes from http://terminology.hl7.org/CodeSystem/v3-MaritalStatus where concept is-a M
  • Include codes from http://terminology.hl7.org/CodeSystem/v3-MaritalStatus where concept is-a U
  • Include codes from http://terminology.hl7.org/CodeSystem/v3-MaritalStatus where concept is-a W
  • Include all codes defined in https://www.ehealth.fgov.be/standards/fhir/CodeSystem/cd-civilstate
